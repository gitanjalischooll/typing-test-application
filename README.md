# Typing Test Application

A Java Swing-based Typing Test Application developed as part of academic learning in Java and Object-Oriented Programming.
A GUI-based typing test application developed in Java using Swing, featuring timed typing challenges and performance evaluation.

## Project Overview

This application helps users practice typing speed and accuracy by displaying a text passage that must be typed within a specified time limit.

The system tracks user performance and provides results based on the typing test.

## Features

* 60-second typing timer
* Random text passages
* User typing area
* Start test functionality
* Submit answers
* Generate new typing passages
* Display typing results
* Simple graphical user interface

## Technologies Used

* Java
* Java Swing
* Object-Oriented Programming
